Pinkman Platformer Game

Game platformer sederhana menggunakan Python dan Pygame

Cara Menjalankan
1. Jalankan file main.py
2. Pastikan semua asset tersedia

Catatan
Disarankan untuk mendownload file ZIP yang berisi asset lengkap, lalu ekstrak dan letakkan dalam satu folder yang sama sebelum menjalankan game
